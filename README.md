# Анализ продаж по категориям (SQL)

В этом репозитории представлен пример SQL-запроса для обработки данных о продажах. 

### Что делает запрос:
* Очищает данные от транзакций с низким чеком.
* Группирует продажи по категориям товаров.
* Фильтрует категории по минимальному порогу выручки.
* Сортирует результат для выявления лидеров продаж.

Этот пример демонстрирует базовые навыки работы с агрегатными функциями, группировкой и фильтрацией данных.
